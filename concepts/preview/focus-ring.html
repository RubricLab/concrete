Rules
Width
Always 3px. Drawn as box-shadow: 0 0 0 3px var(--ring-*). Never outline, which squared-corners things with rounded radii.
Offset
Zero. The ring sits flush against the element edge. Any offset reads as a secondary UI layer and breaks the solid, material feel.
Visibility
Only on :focus-visible, never on :focus. Mouse clicks should not raise a ring.
Tint
Match the element's semantic role — a destructive button rings in --ring-error, an upgrade CTA in --ring-ultra. Most inputs and surfaces use the default sky ring.
Companion
Combine with a one-step border darkening (ink-2 → ink-5) to keep the control legible when the ring is half off-screen.
This primitive documents the tokens. Nothing here owns the values — --ring, --ring-ink, etc. live in the root palette and every other component pulls from them.
