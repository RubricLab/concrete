# Concrete — UI kit

A single product-surface kit that composes the Concrete primitives into a plausible AI-native app shell:

- **Sidebar** — workspace picker, primary nav, thread list, footer account row
- **Header** — breadcrumb, model picker affordance, command palette `⌘K` button, avatar
- **Message turn** — user turn + assistant turn, with a tool-call sub-block and citation chips
- **Composer** — prompt textarea, attach / model / send kbd

All atoms come verbatim from Concrete tokens (`colors_and_type.css`) and match the preview cards in `../../preview/`.

Open `index.html` for the composed view.
